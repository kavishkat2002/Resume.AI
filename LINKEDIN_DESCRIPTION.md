## ✅ Short Version *(Recommended — paste directly into LinkedIn)*

**Resume.AI — AI-Powered Career Optimization Platform**

I built Resume.AI to solve a real problem I kept seeing: talented professionals losing out on jobs not because of their skills, but because their resumes couldn't pass Applicant Tracking Systems (ATS) or failed to connect with hiring managers.

Resume.AI is a full-stack, production-ready web platform that uses Large Language Models (LLMs) and vector-based semantic analysis to turn a plain resume into a targeted, ATS-optimized career asset — in minutes.

**What it does:**
- Scores resumes against any job description in real time and surfaces missing keywords
- Generates narrative-driven cover letters built around the candidate's actual career trajectory
- Profiles GitHub repositories and LinkedIn history to auto-populate a verified skills snapshot
- Provides a 4-Tier Agentic Memory AI chatbot (Short-Term · Semantic · Episodic · Procedural) for hallucination-free career coaching
- Exports polished PDFs, Word documents, and plain-text resumes across Modern, Classic, and Minimal templates

**Tech Stack:**  
React (TypeScript) · Vite · Tailwind CSS · shadcn/ui · Supabase (PostgreSQL + pgvector) · Deno Edge Functions · OpenAI · Google Gemini · OpenRouter · TanStack React Query

The platform is live at **resume-ai.dev** and open-source on GitHub. It is actively used by job seekers across Sri Lanka, the United States, United Kingdom, Australia, Canada, and India — wherever professionals are competing in modern, algorithm-driven hiring pipelines.

## 📄 Extended Version *(for blog posts, portfolio sites, or bios)*

**Resume.AI — Intelligent Career Optimization Platform Powered by LLMs and ATS Intelligence**

In today's hyper-competitive job market, a well-qualified candidate can be filtered out before a human ever reads their resume. Applicant Tracking Systems (ATS) power the first round of screening at over 99% of Fortune 500 companies and the vast majority of SMEs globally. I built Resume.AI specifically to level that playing field.

Resume.AI is a production-grade, full-stack web application that leverages Large Language Models, vector embeddings, and a multi-agent agentic memory architecture to transform static resumes into precision-targeted, ATS-optimized career documents tailored to specific job descriptions.

### Core Features

**🤖 AI Career Advisor Bot — 4-Tier Agentic Memory Architecture**  
Unlike generic chatbots, the Career Advisor Bot uses a four-tier memory model to deliver coherent, factual, hallucination-free coaching:
- *Short-Term Memory* — retains the last 10 conversation turns for fluid dialogue
- *Semantic Memory* — reads the user's live resume data as verified facts
- *Episodic Memory* — tracks past app activity (ATS scores run, profiles analyzed)
- *Procedural Memory (SOPs)* — enforces strict career-guidance workflows to eliminate invented or off-topic responses

Supports OpenAI, Google Gemini, and OpenRouter API keys with automatic provider detection.

**📊 Real-Time ATS Score Analyzer**  
Instantly measures keyword density, parse-ability, and semantic alignment between a resume and a target job description, delivering a scored report with actionable improvement suggestions.

**🔗 GitHub & LinkedIn Profiler**  
Analyses repository languages, contribution graphs, and professional history to auto-generate a verified technical skills profile — eliminating the blank-page problem for developers and engineers.

**📝 Psychological Narrative Cover Letter Generator**  
Moves beyond fill-in-the-blank templates. The engine constructs a 4-paragraph psychological narrative hook anchored in the candidate's real career story, crafted to engage hiring managers emotionally and logically.

**🎨 Multi-Template Resume Builder with Smart Export**  
Supports Modern, Classic, and Minimal layout templates with one-click application from a live gallery. Downloads are auto-named `FirstName_LastName_CV.pdf / .docx / .txt` for professional consistency.

**📈 Skill Gap Analysis & Learning Path Recommendations**  
Identifies missing hard skills for a target role and maps personalized learning resources to close those gaps efficiently.

**⚙️ Application Tracker Dashboard**  
Centralised tracking of job applications, ATS scores, and profile completion — with always-accurate stats pulled from persistent local state.

### Technical Architecture

| Layer | Technology |
|---|---|
| Frontend Framework | React 18 (TypeScript) + Vite |
| UI System | Tailwind CSS + shadcn/ui (Radix Primitives) |
| State Management | TanStack React Query + Context API |
| Database | PostgreSQL via Supabase + pgvector for semantic search |
| Authentication | Supabase Auth (JWT + Row-Level Security) |
| Serverless Runtime | Deno on Supabase Edge Functions |
| AI/LLM Layer | OpenAI GPT · Google Gemini · OpenRouter (free tier) |
| Document Export | jsPDF + docx library |
| Deployment | Vercel (frontend) + Supabase global edge network |
