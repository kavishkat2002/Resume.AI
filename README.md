# Resume.AI - AI-Powered Career Success

Professional AI-driven platform for optimizing resumes and crafting compelling cover letters.

## Features

- **ATS Optimization**: Tailor your resume to specific job descriptions.
- **Narrative Cover Letters**: Transform your background into a human storytelling experience.
- **GitHub & LinkedIn Analysis**: Extract skills and projects directly from your professional profiles.
- **Skill Gap Analysis**: Identify missing keywords and learn required technologies.

## Technology Stack

- **Frontend**: Vite, React, TypeScript, Tailwind CSS, shadcn/ui.
- **Backend**: Supabase Edge Functions (Deno).
- **AI Integration**: Custom LLM gateway with strict narrative enforcement.

## Getting Started

1. **Clone the repository**:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. **Install dependencies**:
   ```sh
   npm i
   ```

3. **Start the development server**:
   ```sh
   npm run dev
   ```

## Deployment

Deploy using Supabase CLI and your preferred hosting provider (Vercel, Netlify, etc.).

```sh
# Deploy Edge Functions
supabase functions deploy
```
